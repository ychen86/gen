# Worksheet Generators

A browser-based math worksheet generator. Pick a course from the cover page, then
generate endless standards-aligned practice, card sorts, and warm-ups — with answer
keys, print/PDF, and Word export.

**Live (beta):** once GitHub Pages is enabled, the site is at
`https://<your-username>.github.io/<your-repo>/`

## Courses

| Course | Status |
| --- | --- |
| Algebra 1 | Coming soon |
| Geometry | Coming soon |
| **Algebra 2** | **Live (beta)** — 7 units · 120 lessons, sequences and functions through statistics |

## Files

```
index.html                     Cover page — loads automatically (the course selector)
worksheet-generator-v64.html   The Algebra 2 generator (self-contained; React bundled in)
.nojekyll                       Tells GitHub Pages to serve files as-is
```

The cover page links to the generator by relative path, so both files must stay in the
same folder.

## Run it

- **Online:** GitHub → **Settings → Pages → Deploy from a branch → `main` / `root`**, then
  open the Pages URL.
- **Locally:** double-click `index.html` (no install or server needed).

## Beta notes

- This is a beta for content testing. If a generated problem reads oddly, a graph looks
  off, or an answer key seems wrong, that's the kind of feedback worth capturing.
- If **Download Word** or **Print** seems to do nothing, allow pop-ups for the page.
- GitHub Pages is public and has no access control — this URL is for testing, not for sale.
